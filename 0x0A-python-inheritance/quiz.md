#### Question #0

What do these lines print?

```python
a = 12
if a > 2:
    if a % 2 == 0:
        print("Holberton")
    else:
        print("C is fun")
else:
    print("School")
```

- [ ] School
- [*] Holberton
- [ ] C is fun
